# Hacker News Reader - Flutter Assignment

A simple Hacker News reader app built with Flutter using the official Hacker News API.  
The app displays top Hacker News stories and allows users to view story details along with first-level comments.

## Assignment Overview

This project was created as part of a Flutter internship assignment.  
The goal was to build a Hacker News reader app using Flutter and the Hacker News API.

## Features

- Displays top Hacker News stories
- Fetches story IDs from the Hacker News top stories API
- Fetches story details using item detail API
- Detail screen for each story
- Displays first-level comments using the `kids` array
- Pull-to-refresh on home and detail screens
- Opens article links in external browser
- Loading, empty, and error states
- Basic Hacker News-inspired UI
- Clean Architecture-based folder structure
- Provider-based state management

## APIs Used

### Top Stories API

```text
https://hacker-news.firebaseio.com/v0/topstories.json
```
